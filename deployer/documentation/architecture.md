# Arquitetura do Deploy Platform

## VisГЈo Geral

O Deploy Platform Г© uma aplicaГ§ГЈo web para automaГ§ГЈo de deploys de projetos Docker. Foi projetado com os seguintes princГӯpios:

- **Zero dependГӘncias externas** вҖ” apenas mГіdulos nativos do Node.js
- **ConfiguraГ§ГЈo dinГўmica** вҖ” serviГ§os definidos via JSON, nГЈo hardcoded
- **Streaming em tempo real** вҖ” logs via Server-Sent Events (SSE)
- **Stateless** вҖ” estado persistido em arquivos JSON simples
